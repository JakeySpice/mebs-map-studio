One-Page Formulation

The deliverable is a document
A thinking tool — not a BSP generator. Data stays in this browser.

1About & strengths

The person opens the page — start with who they are at their best.

Consider using a pseudonym if this document may travel.

2What a good life looks like

Goals in the person's own terms — one per line.

3What's happening

Describe plainly and briefly — this never dominates the page.

4Why we think it happens

Hypotheses, one per line. The page frames each one cautiously — guesses, held lightly.

5What we'll try

One idea per line in each group.

6How we'll know

What we'll watch, and when we'll look again.
