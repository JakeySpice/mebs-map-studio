MEBS Minimal Lab ✿

Seven off-shoots of MEBS Map Studio, each stripped to a different irreducible core of the product — and nothing else. Open each one, work the same case, and notice which essentials you actually reach for.

All seven load the same de-identified sample case — Alex (pseudonym) — and each stores its data separately in this browser. Every variation is a single HTML file.
